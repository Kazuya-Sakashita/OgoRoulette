# z-indexが効かない？スタッキングコンテキストの落とし穴

## 「z-index上げたのに、なぜか後ろに隠れる…」

OgoRouletteで、こんなバグがありました。

シェアボタンを押すと確認ダイアログが出るはず… なのに出ない。ボタンを押しても何も起きないように見える。

コードを確認すると、確認ダイアログのz-indexは `z-10`。シェアシートの背景は `z-80`。

**z-10 < z-80 なので、ダイアログが背景の裏に隠れていた。**

---

## z-index の基本

`z-index` は「重なり順」を指定するCSSプロパティです。数値が大きいほど手前に表示されます。

```css
.bottom { z-index: 1; }  /* 奥 */
.top    { z-index: 9; }  /* 手前 */
```

ただし、これには大事な前提があります。**同じスタッキングコンテキスト内での比較** です。

---

## スタッキングコンテキストって何？

「スタッキングコンテキスト（stacking context）」は、z-indexの比較が行われる「グループ」のことです。

日常の例で言うと：

> 会社の建物が2棟あります。A棟の7階とB棟の3階、どちらが上ですか？

この比較は「どの棟か（コンテキスト）」が違うので、フロア番号だけでは答えられません。

CSSのz-indexも同じです。**別のスタッキングコンテキストに属する要素は、z-indexで比較できません。**

---

## どんなときにスタッキングコンテキストが作られる？

- `position: fixed` または `position: sticky`
- `opacity` が1未満
- `transform`, `filter`, `clip-path` などが指定されている
- `z-index` が `auto` 以外で、`position` が `relative`/`absolute` である

Framer Motionの `motion.div` はアニメーションのために `transform` を内部で使うことがあります。これがスタッキングコンテキストを作ることがあります。

---

## OgoRouletteでのバグ

シェアシートの構造はこうなっていました：

```tsx
{/* シェアシート本体：fixed + z-80 */}
<motion.div className="fixed inset-0 z-80 ...">
  {/* シェアシートの中身 */}
</motion.div>

{/* 確認ダイアログ：absolute + z-10 */}
{showConfirm && (
  <motion.div className="absolute inset-0 z-10 ...">
    {/* ダイアログの中身 */}
  </motion.div>
)}
```

`absolute inset-0` は「親要素を基準に配置」という意味です。しかし `z-10` は `z-80` のシェアシートとは**別のスタッキングコンテキスト**として扱われます。

結果：ダイアログがシェアシートの裏に隠れる。

---

## 修正方法

```tsx
{/* 修正前 */}
<motion.div className="absolute inset-0 z-10 ...">

{/* 修正後 */}
<motion.div className="fixed inset-0 z-[90] ...">
```

**2つの変更：**

1. `absolute` → `fixed`：画面全体を基準に配置する（親要素のコンテキストに依存しなくなる）
2. `z-10` → `z-[90]`：シェアシート（z-80）より確実に手前に来る数値にする

`fixed` にすることで、どのスタッキングコンテキストに属していても、画面の最前面近くに表示できます。

---

## まとめ

z-indexが「効かない」ときのチェックリスト：

- [ ] 比較したい要素は同じスタッキングコンテキスト内にあるか？
- [ ] 親要素が `transform` や `opacity` を持っていないか？
- [ ] `position` は指定されているか？（`z-index` は `position: static` では効かない）
- [ ] `fixed` にすることでコンテキストをリセットできないか？

z-indexのデバッグは「数値を大きくしても効かない」という状況から始まることが多いです。そういうときは数値ではなく **コンテキスト** を疑いましょう。
