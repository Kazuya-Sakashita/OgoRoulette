# OgoRoulette ブログ記事

OgoRoulette の開発を通じて書いたブログ記事をカテゴリ別に管理しています。

## ディレクトリ構成

| ディレクトリ | 内容 |
|-------------|------|
| [dev-story/](dev-story/) | OgoRoulette の開発ストーリー連載。設計の意思決定・auth 実装・セッション管理・録画機能など、プロジェクト固有の記録。 |
| [tech/](tech/) | 汎用技術記事。JavaScript 言語仕様（TDZ・非同期 setState）、モバイル CSS（dvh・overflow・z-index）、セキュリティなど、他プロジェクトにも転用できる知見。 |
| [ai-driven-dev/](ai-driven-dev/) | AI 駆動開発のメソドロジー・事例・プロンプト集。Claude Code の活用進化、コンテキスト管理、Lighthouse 改善、ChunkLoadError 調査など。 |
| [vibe-coding-hackathon/](vibe-coding-hackathon/) | Vibe Coding ハッカソン参加記（全 5 話）。AI と一緒に開発する体験を中心にまとめた連載。`draft_` プレフィックスは同シリーズの別バージョン原稿。 |

## 命名規則

- `NN_タイトル.md` — 番号付き連載記事（dev-story / tech）
- `NN-kebab-case.md` — 技術記事（ai-driven-dev 直下）
- `draft_NN_タイトル.md` — 公開前の原稿または別バージョン
