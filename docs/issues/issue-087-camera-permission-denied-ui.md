# ISSUE-087: カメラ権限拒否・未対応 UI

**ステータス:** 実装済み
**優先度:** Medium
**前提:** ISSUE-085 完了（ISSUE-086 と同時実装）

## 概要

カメラ権限を拒否された場合・カメラが存在しない端末の場合の
エラー UI と誘導テキストを実装する。

## 実装内容

**ファイル:** `components/qr-scanner.tsx` 内エラー状態 UI

- `NotAllowedError`: 「カメラへのアクセスが許可されていません。設定アプリから許可してください。」
- `NotFoundError`: 「カメラが見つかりません。コード入力からご参加ください。」
- スキャンタブ選択時のみ `getUserMedia` を呼ぶ（ページ初期表示で権限を要求しない）
- エラー状態では再試行ボタン（`NotAllowedError` 以外）または手動入力誘導リンク
