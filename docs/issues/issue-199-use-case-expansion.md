# 使用シーン拡大訴求（奢り以外の用途 LP・ホーム強化）

## 背景

G-STACK 評価「Goal（目的の明確さ）」-3点の根拠。
OgoRoulette は「誰が奢るか決めるアプリ」として認知されているが、
実際の用途は「順番決め」「担当決め」「罰ゲーム選出」「じゃんけん代替」等
多岐にわたる。しかし LP・ホーム画面の訴求が「奢り」に特化しすぎており、
潜在ユーザーの流入機会を失っている。

例えば検索クエリ「ランダム 決める アプリ」「順番 決め方 スマホ」では
現在の訴求では刺さらない。

## 問題

- LP・ホーム画面の訴求が「飲み会のおごり」のみ
- 「じゃんけん代わりに使う」「担当を決める」ユーザーが発見できない
- how-to-use も奢りシナリオ限定
- ルーレットの汎用性が伝わっていない

## 目的

- 使用シーンの多様性を訴求してユーザー獲得の天井を引き上げる
- G-STACK Goal スコアを 17 → 19 に改善する
- SEO キーワードの幅を広げる

## 対応内容

### LP の「SCENES」セクション強化

現在のシーン表示（飲み会・社内ランチ・打ち上げ・学生グループ）に加え、
「奢り以外の使い方」セクションを追加。

```
使い方いろいろ:
🎯 "順番決め"  — 発表順・掃除当番・プレゼン順を公平に
🎮 "罰ゲーム"  — ゲームの罰当て・ドリンクの種類決め
📋 "担当決め"  — 当番・役割・リーダーをフェアに
🎁 "プレゼント" — 誰がサプライズ幹事かを演出で発表
```

### ホーム画面のタイトル・説明文の汎用化

```typescript
// 現在: 「おごりをルーレットで決めよう」
// 変更後: 「ルーレットで、今日の運命を決めよう」
// サブコピー: 奢り・順番・担当・罰ゲーム — なんでも公平に
```

### メタデータ（SEO）の強化

```typescript
// app/layout.tsx または app/lp/page.tsx
keywords: [
  "おごり決める",
  "ルーレット アプリ",
  "順番決め アプリ",
  "担当決め スマホ",
  "罰ゲーム 決め方",
  "じゃんけん 代わり アプリ",
  "飲み会 誰が払う",
  "グループ ランダム 選出",
]
```

### how-to-use ページのシナリオ追加

「STEPS」セクションに「奢り以外の使い方」カードを追加。
