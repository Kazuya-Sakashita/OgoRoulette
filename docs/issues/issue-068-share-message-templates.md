# ISSUE-068: SNS シェアメッセージテンプレートの追加

## ステータス
✅ 完了

## 優先度
**Medium**

## カテゴリ
SNS / Share / Copywriting

## 概要
「定番」の 1 種類のみだったシェアテキストを、状況に応じた 5 種類のテンプレートに拡充する。
ShareSheet 上のテンプレートチップで切り替え・手動編集も可能にする。

## 実装したテンプレート

| ID | ラベル | 用途 |
|----|--------|------|
| `classic` | 定番 | デフォルト。シンプルで使いやすい |
| `dramatic` | ドラマチック | 長文。物語性があり拡散しやすい |
| `roast` | いじり | 友達へのいじりトーン。面白系 |
| `bill` | 金額付き | 支払い額を明示。金額情報がある時の自動選択 |
| `group` | グループ | 参加人数を含む。大人数グループ向け |

### テンプレート例（winner = "田中"）

**定番**:
```
🎰 OgoRouletteで田中さんが奢りに決定！ #OgoRoulette
```

**ドラマチック**:
```
運命のルーレットが回った… 止まった先には 田中 さんの名前が！🎡✨ 今日は 田中 さんのおかげで最高のランチ確定！ #OgoRoulette #おごり
```

**いじり**:
```
田中 さん、ついに来ましたね😏🎰 神のお告げです。本日の奢り担当は 田中 さんに決定しました。ご清算をお願いします🙏 #OgoRoulette
```

**金額付き（¥5,000 奢り）**:
```
🎰 田中さんが奢り担当に！ ¥5,000をご負担いただきます🙏 ごちそうさまです！ #OgoRoulette
```

**グループ（5人）**:
```
🎰 5人の中から田中さんが今日のスポンサーに選ばれました！ 太っ腹な田中さん、ごちそうさまです🎉 #OgoRoulette
```

## UI
- ShareSheet にテンプレートチップ（ピル形）を追加
- 選択中チップは winnerColor で着色
- テキストプレビュー欄右上の鉛筆アイコンで自由編集
- 編集後はテンプレート再選択でリセット
- 文字数カウンター（編集時のみ表示、280 字上限）

## 実装ファイル
- `lib/share-service.ts` — `SHARE_TEMPLATES: ShareTemplate[]`
- `components/share-sheet.tsx` — テンプレート選択 UI + テキスト編集欄
