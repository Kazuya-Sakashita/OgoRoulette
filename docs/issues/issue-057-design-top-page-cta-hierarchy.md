# ISSUE-057: トップページの CTA 階層が不明確・ボタンテキストに不統一

## ステータス
✅ 完了 — 2026-04-06

## 優先度
**High**

## カテゴリ
Design / UX

## 概要
トップページ（`/`）に4つの主要 CTA（Google・LINE・X・まず試してみる）が並んでいるが、視覚的な重み付けがなく「どれを押すべきか」が伝わらない。さらに、ボタンのラベル文体が「〜でログイン」「〜で続ける」と混在しており、ブランド一貫性を損なっている。

## 問題の詳細

### 1. CTA 階層の欠如
- Google・LINE・X・「まず試してみる」の4ボタンがほぼ同サイズで縦に並ぶ
- ユーザーが「どれを選ぶべきか」を判断する手がかりがない
- SNS ログインが「アカウント保存のメリットがある選択肢」、ゲストが「手軽に試す選択肢」という意図が見た目に反映されていない
- ログインボタン群とゲストボタンの間のセパレーターは細く、視覚的分離が弱い

### 2. ボタンテキストの不統一
| ボタン | 現在のテキスト |
|--------|--------------|
| Google | Googleでログイン |
| LINE | LINEでログイン |
| X | Xで続ける |
| ゲスト | まず試してみる |

「〜でログイン」と「〜で続ける」が混在。X だけ異なるフレーミングになっている。

### 3. 副テキストの弱さ
- SNS ログイン下の「保存・共有・履歴が使えます」は全体への説明文だが、Google ボタンの直下にのみ配置されており Google の補足説明と誤読される
- ゲスト下の「ログイン不要で体験できます」は有益だが、上記の構造問題で埋もれている

## 改善案

### テキスト統一（最小変更）
```
Google でログイン  →  Google で始める
LINE でログイン    →  LINE で始める
X で続ける        →  X で始める
まず試してみる     →  ログインせずに試す（or 現状維持）
```

### 視覚階層の改善
1. SNS ログインボタン群を「アカウントを作成」セクションとして囲む、または見出しを付ける
2. 「まず試してみる」をアウトライン（現状）のまま維持し、サイズかコントラストを落として副次的選択肢であることを示す
3. 副テキスト「保存・共有・履歴が使えます」を Google ボタン下ではなく3つの SNS ボタン全体の上に移動

## 影響範囲
- `app/page.tsx` — ボタンテキスト・レイアウト変更
- デザイントークンの変更なし（テキスト・余白のみ）
