# ISSUE-140: ランディングページにプライバシー安心感テキストを追加する

## 概要

`app/page.tsx` のソーシャルプルーフバッジの一部として「🔒 本名は公開されません」を表示し、
初訪問者が個人情報への不安なく参加できるよう安心感を提供する。

---

## 背景

- ルーレットアプリへの参加に「本名がバレるのでは？」という心理的障壁がある
- 特に会社の飲み会・合コンなど、面識が薄い相手とのグループ利用時に顕著
- ISSUE-137（ソーシャルプルーフ）と組み合わせて1行のバッジエリアに収める

---

## 修正内容

ISSUE-137 と同一の実装ブロック内に含まれる：

```tsx
<div className="flex items-center gap-1.5 text-xs text-muted-foreground/70">
  <span>🔒</span>
  <span>本名は公開されません</span>
</div>
```

「公開名」機能（ISSUE-106で実装）の存在をランディングで初めて伝える接点として機能する。
