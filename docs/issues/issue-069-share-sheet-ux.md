# ISSUE-069: ShareSheet UX 改善 — テンプレート選択 + テキスト編集

## ステータス
✅ 完了

## 優先度
**Medium**

## カテゴリ
Design / UX / Share

## 概要
ShareSheet の UI を改善し、メッセージテンプレート選択とテキスト自由編集を追加する。
「シェアする前に一言添えたい」「状況に合ったメッセージにしたい」ニーズに応える。

## 問題の詳細（改善前）

- シェアテキストが固定の 1 種類のみで変更不可
- テキストが画面に表示されないため、何がシェアされるか不明
- ユーザーが投稿前に確認・調整できる場所がない

## 実装した改善

### 1. テンプレートチップ
- ピル型のチップを横並びで表示
- 選択中チップ: winnerColor で背景着色 + シャドウ
- 非選択チップ: `bg-white/8` の半透明

### 2. メッセージプレビュー
- テンプレート選択ごとにリアルタイムでテキスト更新
- 右上の鉛筆アイコンで編集モードへ

### 3. 自由テキスト編集
- `<textarea>` で自由編集
- 文字数カウンター（`{n}/280`）を表示
- 「完了」ボタンで確定（チェックアイコン）
- テンプレート再選択でカスタムテキストをリセット

### 4. プラットフォームボタンの説明補強
- X / LINE ボタンに `title="テキストのみ（動画なし）"` を追加
- Primary CTA は「動画をシェア」に変更（従来:「SNSでシェア」）

## 影響範囲
- `components/share-sheet.tsx` — 全面刷新
