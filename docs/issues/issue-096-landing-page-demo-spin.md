## 概要

現在の `/`（ウェルカムページ）はログインボタンのみで、
初見ユーザーが「OgoRouletteで何ができるか」を体験する手段がない。

「百聞は一見に如かず」— ログイン前に実際にルーレットを回せる
デモセクションを追加することで、コンバージョン率を高める。

---

## 実装内容

### デモルーレットの仕様

- デフォルト名: `さくら`, `たろう`, `はな`（変更不可、シンプルに固定）
- 「🎰 回す」ボタンをタップするとアニメーション開始
- 約1.5秒かけて徐々に減速し、ランダムな1名を選択
- 当選者名を表示後、「無料で使ってみる →」CTA を表示
- DB・ルーム作成なし、完全クライアントサイド

### 配置

ロゴ・タグラインの直下、ログインボタン群の上に挿入。
ファーストビューで体験できることを優先。

### アニメーション

```typescript
// 徐々に減速するアニメーション
const delay = step < 12 ? 80 : step < 16 ? 160 : 300
```

## 実装箇所

## 受け入れ条件

- 初見ユーザー（localStorage に `ogoroulette_visited` がない）がデモを体験できる
- 「回す」ボタンでアニメーション → 当選者表示 → 「無料で使ってみる →」CTA
- CTA は `/home` に遷移し `ogoroulette_visited` をセット（ゲストモード）
- SSR/クライアント不一致エラーなし
- 既存ログイン UI に影響なし
