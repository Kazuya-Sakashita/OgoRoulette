# ISSUE-086: /scan ページへの QrScanner 統合

**ステータス:** 実装済み
**優先度:** High
**前提:** ISSUE-085 完了

## 概要

`/scan` ページのQRスキャンモードのプレースホルダーを削除し、
`QrScanner` コンポーネントを統合する。

## 実装内容

**ファイル:** `app/scan/page.tsx`

- プレースホルダー（カメラアイコン + ダッシュ枠）を削除
- `<QrScanner onScan={(code) => router.push(\`/join/\${code}\`)} />` に置き換え
- スキャンモード選択時のみ QrScanner を active にする（ページ開いた瞬間に権限要求しない）
- 権限拒否 / カメラなしの場合はエラーメッセージ + 手動タブへのリンク
- 5秒間スキャン未検出でコード入力タブへの誘導バナー表示
