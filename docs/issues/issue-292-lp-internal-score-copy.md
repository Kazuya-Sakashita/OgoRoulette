# ISSUE-292: Low — LP に内部スコア（「セキュリティスコア 99/100」）が公開されており外部ユーザーに伝わらない

## ステータス
🔲 未対応 — 実機確認で発覚。LPページに「セキュリティスコア 99/100」「セキュリティ監査で全評価軸 PASS 取得済み」という記載があるが、これは内部評価フレームワーク（G-STACK/HEART等）のスコアであり、一般ユーザーには全く意味が伝わらない。

## 優先度
**Low / コピーライティング / 信頼性**

## カテゴリ
Copy / Marketing / LP / Credibility

---

## 問題

一般ユーザーが「セキュリティスコア 99/100」を見ても：
- どこが評価したのか不明
- 何を基準にした 99/100 なのか不明
- 「自称 99/100」に見えて逆に信頼性を下げる

また LP に「🔥🎰🔥」のような過剰な絵文字演出が散見され、
全体的にセルフプロモーション感が強い。

---

## 修正方針

### 削除または具体的な内容に置き換える

```
Before: 「セキュリティスコア 99/100 取得済み」
After:  「個人情報を収集しません」または「ログイン不要・名前以外の情報不要」
```

### LP の「信頼性」表現を具体化する

```
Before: 「全評価軸 PASS 取得済み」
After:  「本名・メールアドレス不要」「データは暗号化通信で保護」
```

---

## 受け入れ条件

- [ ] 内部スコア参照の記述をLPから削除または具体的なユーザー価値に置き換えること
- [ ] 一般ユーザーが読んで信頼性が上がる内容になっていること
